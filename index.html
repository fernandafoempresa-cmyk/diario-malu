Usuários e Acesso

Gerencie quem acessa o diário da Malu

💡 Para acesso na nuvem (qualquer dispositivo): configure o Firebase no arquivo HTML.

⚠️ Regras do Firestore (obrigatório):
No console Firebase → Firestore → Regras, cole o código abaixo e clique em Publicar:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /malu/data { allow read: if request.auth != null; allow write: if request.auth != null && get(/databases/$(database)/documents/malu/users/$(request.auth.uid)).data.approved == true && get(/databases/$(database)/documents/malu/users/$(request.auth.uid)).data.role in ['admin','editor']; } match /malu/users/{uid} { allow read: if request.auth != null; allow write: if request.auth != null && ( request.auth.uid == uid || get(/databases/$(database)/documents/malu/users/$(request.auth.uid)).data.role == 'admin' ); } } }
Apenas administradores podem gerenciar usuários.
